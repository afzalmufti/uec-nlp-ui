Natural Language Processing, voice user interface prototype

This prototype aims to use conversational AI to automate the identification of a caller while in a queuing scenario waiting to be connected to a call handler.

Instructions

  1. Dial 0113 123 456 to access the voice user interface.
  2. You will be asked a series of questions, if you are calling about yourself or somebody else, your name and date of birth.
  3. The voice model is trained on a pre-populated database of xx entries of which the NLP model will attempt to match the submitted details to. If a match is found, the NHS number will be read back to you to illustrate this match. Some test data entries are listed below for you to try.
  4. Once completed, you may proceed to the call handler interface demo to retrieve the call from the queue with demographic details populated or an option to correct the name or date of birth.
